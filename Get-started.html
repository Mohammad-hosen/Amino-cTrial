Welcome to Amino

Simple steps of getting started with Amino.

1. Add the extension to your browser

ADD TO CHROME

To learn more about the permissions we request at install, see Permissions.

2. Open the editor from the toolbar icon

For demonstration purposes, open it while you're on this page.

3. Alternatively, open the editor from within DevTools

Open dev tools:

  • Mac: Command+Option+I
  • Windows/Linux: F12 or Control+
    Shift+

Select the Elements tab:

Then, select the Amino sub-tab:

4. Begin composing your CSS

For demonstration purposes, add this CSS to the the editor and save it while on this page:

        body {
        font-family: Comic Sans MS;
        }

That's it

That's the gist of how to use Amino to customize sites with CSS.
